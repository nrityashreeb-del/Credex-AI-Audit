# Credex AI Spend Auditor 🚀

This is a technical assessment for the **Credex Web Development Internship**. It is a functional product designed to help organizations audit their AI tool spending and identify immediate savings.

## 🛠️ The Problem
Many companies have "Shadow AI" spending—multiple employees paying for individual $20/month Pro accounts instead of using consolidated Team plans or enterprise licenses, leading to 15-30% cost leakage.

## 💡 The Solution
The **AI Spend Auditor** calculates potential savings based on team size and current monthly spend, providing users with an immediate "Savings Report" and a call-to-action to book a Credex optimization.

## 📂 Required Deliverables
- **Audit Engine:** Built with Vanilla HTML5/CSS3 and JavaScript.
- **DEVLOG.md:** Daily sprint logs and technical decisions.
- **ECONOMICS.md:** Analysis of how this tool serves as a lead-generation engine.
- **ARCHITECTURE.md:** Overview of the application structure and logic.
- **PRICING_DATA.md:** Verified links to official vendor pricing (OpenAI, Anthropic, Google).
